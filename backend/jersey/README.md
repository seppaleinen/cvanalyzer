# Jersey

* Connecting to database
* [Apache Tika - Parsing different document types like (doc, docx, pdf...)](https://tika.apache.org/1.8/examples.html)
* Providing rest-api

Implemented now with [Jersey](https://jersey.java.net/)

To compile
```
mvn clean install
```

To run server
```
java -jar target/jersey.jar
```
